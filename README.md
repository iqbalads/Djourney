# D'Journey of West Java - Internship Project

## 📌 Project Overview
**D'Journey** adalah platform portal aplikasi perjalanan yang dirancang untuk memberikan kemudahan bagi wisatawan dalam menemukan destinasi wisata, kuliner, dan acara (events) di Jawa Barat. Proyek ini dikembangkan selama masa magang di **PT Jaswita Jabar**.

Aplikasi ini berfungsi sebagai panduan digital komprehensif bagi pengguna untuk menjelajahi keindahan Jawa Barat dengan fitur pemesanan paket wisata (Open Trip & Private Trip) serta informasi artikel terkini.

> **Note:** Repositori ini bersifat **Dokumentasi Saja** dikarenakan kebijakan privasi perusahaan (PT Jaswita Jabar). Kode sumber tidak dipublikasikan secara terbuka.

---

## 🚀 Fitur Utama (Features)

Sesuai dengan platform web dan mobile-nya, D'Journey memiliki fitur-fitur berikut:

- **Destination Discovery:** Menampilkan berbagai objek wisata populer, tersembunyi, dan bersejarah di Jawa Barat.
- **Trip Management:**
  - **Open Trip:** Pengguna dapat bergabung dengan grup perjalanan yang sudah terjadwal.
  - **Private Trip:** Layanan perjalanan eksklusif yang dapat disesuaikan dengan kebutuhan pengguna.
- **Culinary & Gastronomy:** Rekomendasi tempat makan dan kuliner khas daerah di Jawa Barat.
- **Event Calendar:** Informasi mengenai acara budaya, festival, dan kegiatan menarik lainnya di wilayah Jawa Barat.
- **User Profile & Activity:** Manajemen akun, riwayat transaksi, ulasan destinasi, dan fitur simpan (saved) artikel/destinasi.
- **Integrated Payment:** Dukungan pembayaran digital untuk pemesanan tiket dan paket wisata (via Midtrans).

---

## 🛠️ Tech Stack

Berdasarkan implementasi proyek, teknologi yang digunakan meliputi:

- **Mobile Framework:** React Native (Cross-platform Android & iOS)
- **Backend:** Express.js
- **Database:** MySQL
- **State Management & Tools:** Redux / Context API
- **Maps & Location:** Google Maps API
